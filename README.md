https://romantic-wescoff-efc16b.netlify.com/

## Project setup
### Setup
```sh
yarn install
```

### Compiles and hot-reloads for development
```sh
yarn demo
```

### Compiles and minifies for production
```sh
yarn build
```

### Compiles demo for production
```sh
yarn demo:build
```

### Run tests
```sh
yarn test
```

### Lints and fixes files
```sh
yarn lint
```
